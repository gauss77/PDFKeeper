Document Capture Queue File Operations

From the Document Capture Queue, PDF documents can be renamed and deleted.

To Rename a PDF document, perform the following:

  1. From the Document Capture Queue list box, select the PDF document. If the wrong document was selected, select Deselect or press the Esc key.
  2. Select Rename or use the Keyboard shortcut Alt+N.
  3. Enter the new file name, and then select OK or press the Enter key.

To Delete a PDF document from the queue, perform the following:

  1. From the Document Capture Queue list box, select the PDF document. If the wrong document was selected, select Deselect or press the Esc key.
  2. Select Delete or use the Keyboard shortcut Alt+D.
  3. When prompted, select Yes or press Enter to perform the delete.

Note

  • It is NOT recommended to configure the Windows Recycle Bin to delete files immediately as files deleted using Delete will be moved to the Recycle Bin.
