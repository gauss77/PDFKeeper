User FAQ

General Information

What is PDFKeeper?
It is a free, open source PDF document storage system that provides storage, indexing, and retrieval using the free Oracle Database Express Edition (Oracle Database XE).
Who should use PDFKeeper?
Anyone who wants to implement a low cost PDF document storage system in a small or home office. It can be implemented for use on a single user system or in a workgroup.

Software Compatibility

Does PDFKeeper work on other Oracle databases?
Only Oracle Database Express Edition is supported. However, if you are a developer, experienced with both Oracle Database and VB.NET, you should be able to get it to function with some modification on any edition of Oracle Database 10g or later with Oracle Text enabled.
Will other databases be supported in the future?
It would be ideal to expand the scope of PDFKeeper to work with other editions of the Oracle Database and Oracle Database Cloud. However, this will be dictated by need, time permiting.
Will PDFKeeper function on 64-bit versions of Windows?
PDFKeeper is a 32-bit application that should function on 64-bit Windows. However, it has not been tested on a 64-bit version of the Windows operating system.
